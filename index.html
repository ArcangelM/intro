Como aplicar Bootstrap a nuestra pagina web

¿Que es Bootstrap?

Bootstrap es un Framework con el cual buscaremos organizar el contenido de nuestra pagina web de una manera mucho mas atractiva. Una pagina web mucho mejor tanto visual como estructuralmente.

¿Que mas usaremos?

JQuery, el cual hace mas facil el uso de JavaScript en nuestra pagina web pero que ademas es usado por unos componentes de JavaScript de Bootstrap

Empecemos...

Aplicando Bootstrap

Dado que bootstrap cuenta con una gran cantidad de componentes, que pueden ser vistos en su documentacion en linea, aqui solo expondremos la manera como puede ser distribuida la estructura en Bootstrap.

Lo mas importante que debemos conocer es la grid de bootstrap dado que apartir de esta toma forma la estructura de nuestra pagina web. Esta grid cuenta con doce columnas, a nuestro gusto, separaremos en las unidades que necesitemos y siempre teniendo en cuenta que la suma de esas separaciones debe ser igual a doce y ademas en cualquier estructura "contenedor" podremos añadir filas y columnas (div's, section's, article's, aside's, etc). Otro componente que tiene Bootstrap es el de la barra de navegacion, la cual resulta util para acceder por medio de hipervinculos a otras secciones de nuestra pagina web. Su adicion ademas resulta facil y la estructura propuesta por el equipo de Bootstrap es esta:

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#referenciaElemento1">Elemento1 <span class="sr-only">(current)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#referenciaElemento2">Elemento2</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#referenciaElemento3">Elemento3</a>
      </li>
      <li class="nav-item">
        <a class="nav-link disabled" href="#referenciaElemento4">Elemento 4</a>
      </li>
    </ul>
  </div>
</nav>

En la documentacion de Bootstrap se hace mas enfasis en otros modos de mostrar la Navbar sin uso de listas sino con clases, nuevamente se invita al lector mira la documentacion oficial.

En el ejemplo que seguiamos se aplico este modelo de Navbar con la opcion de que cuando se clickease alguno de los elementos, este nos llevara a un de los "aside". Se hace enfasis en algunas funcionalidades de bootstrap como lo es el button class="navbar-toggler" que permite ocupar horizontalmente toda la pagina para editar el tamaño de los botones conforme se añaden mas. El detalle en la etiqueta "span" simplemente para hacer referencia a que se pueden añadir iconos y demas detalles a cada boton posteriormente. Por ultimo se muestra en ul class="navbar-nav" que los botones son elementos de esa lista.

Si queremos hacer referencia a algun elemento de nuestra pagina web en algun boton de la navbar agregamos la opcion id="nombreReferencia" a ese elemento y lo asociamos en la navbar.
